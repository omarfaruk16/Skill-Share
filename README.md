## 📌 Project Purpose
Skill share is a platform where multiple skills are in one place. User can book any skills and start learning. Our top rated instructors and well reputation ensure you for a better experience

---

## 🚀 Key Features

- 1. User Authentication (Email/Password & Google Login)
- 2. Password Reset via Email
- 3. User Profile Page
- 4. Update Profile (Name & Photo)
- 5. Responsive UI Design
- 6. Toast Notifications for user actions
- 7. logged in user can see skill details and book them
